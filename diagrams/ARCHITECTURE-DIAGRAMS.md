## Service Communication Flow

### 1. User Authentication Flow
```
Client вҶ’ API Gateway вҶ’ Auth Service вҶ’ Auth DB
                          вҶ“
                      Generate JWT
                          вҶ“
                    Return Token to Client
```

### 2. Create Note with Images Flow
```
Client вҶ’ API Gateway вҶ’ Notes Service в”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”ҖвҶ’ Media Service вҶ’ File System
                          вҶ“                          вҶ“
                       Notes DB              Store Image & Return ID
                          вҶ“
                    Save Note with Media IDs
```

### 3. Delete Label Flow (Distributed Transaction)
```
Client вҶ’ API Gateway вҶ’ Labels Service в”Җв”Җв”Җв”Җв”ҖвҶ’ Notes Service
                          вҶ“                      вҶ“
                       Labels DB        Remove Label from Notes
                          вҶ“                      вҶ“
                    Delete Label            Update Notes DB
```

### 4. Copy Note Flow
```
Client вҶ’ API Gateway вҶ’ Notes Service в”Җв”Җв”Җв”Җв”Җв”ҖвҶ’ Media Service
                          вҶ“                      вҶ“
                    Find Source Note      Copy Each Image
                          вҶ“                      вҶ“
                    Create New Note       Return New Media IDs
                          вҶ“
                    Save to Notes DB
```
